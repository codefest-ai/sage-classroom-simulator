Why is everyone showing 65%?

The live Zoom path is working. Right now, the prototype is receiving similar live signals for everyone in the room, so it is rendering a shared baseline rather than sharply differentiated individual scores.

What this proves right now

  • The system is ingesting Zoom meeting activity during the session, not only after class.
  • The live dashboard is producing a real classroom state from current meeting signals.
  • This is a prototype-level live signal view, not a finished production analytics model.

What would create stronger differentiation

  • More granular live event channels
  • Chat activity, raised hands, reactions, speaking telemetry
  • More time and more varied interaction patterns
